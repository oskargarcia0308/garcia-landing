QUICK START GUIDE:
==================

STEP 1: UPLOAD TO YOUR WEBSITE
- Log into your web hosting (GoDaddy, Bluehost, etc.)
- Go to File Manager or FTP
- Upload entire folder to public_html/
- Visit: yourwebsite.com/index.html

STEP 2: CONNECT YOUR FORM
- The form currently shows an alert
- To collect real leads, use Formspree.io (free)
- Instructions: formspree.io/setup

STEP 3: SET UP EMAIL SEQUENCE
- Open email-sequence.md
- Copy emails into Mailchimp/ConvertKit
- Set automation trigger: form submission

STEP 4: TEST EVERYTHING
- Submit a test form
- Check you receive the email
- Verify thank you page displays
